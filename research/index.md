Five structured tests on the [MUSDB18-7s](https://zenodo.org/record/1117372) benchmark dataset, run locally on Apple M4 using MPS acceleration. Each test is designed to answer a specific practical question about [HTDemucs](https://github.com/facebookresearch/demucs) stem separation performance.

All raw data is available in the [site repository](https://github.com/stemsplitter/stemsplitter.github.io/tree/main/_data/research). Methodology details are on each individual test page.
